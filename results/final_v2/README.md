# Final V2 Results

Esta carpeta recoge solo los resultados finales utiles para documentar y revisar la version definitiva del proyecto: `PPO_gated_full` (`full_gated_v2`).

Archivos incluidos:

- `global_metrics_all_strategies.csv`: comparativa global de `PPO_gated_full` frente a los baselines clasicos en `version_2`.
- `window_metrics_all_strategies.csv`: metricas por ventana temporal para todas las estrategias comparadas en `version_2`.
- `window_metrics_ppo_gated_full.csv`: metricas por ventana temporal solo del modelo final.
- `portfolio_series_all_strategies.csv`: series temporales de valor de cartera por fold para `version_2`.
- `ppo_gated_full_summary.csv`: resumen agregado del modelo final.
- `ppo_gated_full_fold_results.csv`: resultados completos por fold del modelo final.

No se incluyen datos raw, logs, checkpoints ni figuras de memoria.
